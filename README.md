**Learn C++ Programming**

A comprehensive collection of C++ source files covering fundamental to advanced programming concepts, designed for learning and practice.

**Overview**

This repository serves as a structured learning resource for mastering C++ programming. It contains a variety of scripts ranging from basic syntax and data types to Object-Oriented Programming (OOP) principles, recursion, and file handling.

**Repository Purpose**

The primary goal of this repository is to provide practical code examples for:

- Practicing C++ syntax and core logic.
- Understanding memory management through pointers.
- Implementing Object-Oriented Programming (OOP) using classes and constructors.
- Developing problem-solving skills through small games and mathematical algorithms.

**Key Concepts Demonstrated**

- **Basic Syntax & Data Types**: Usage of int, float, double, and const.
- **Control Flow**: Implementation of if-else statements, for loops, while loops, and do-while loops.
- **Functions & Recursion**: Creating modular code and solving problems like factorials using recursive calls.
- **Arrays & Strings**: Handling one-dimensional and multi-dimensional arrays, as well as string manipulation.
- **Pointers**: Understanding memory addresses and pointer-to-pointer logic.
- **Object-Oriented Programming**: Defining classes, access specifiers (private/public), methods, and constructors.
- **Data Structures**: Using struct and typedef for custom data types.

**Technologies Used**

- **Language**: C++.
- **Standard Library**: &lt;iostream&gt;, &lt;string&gt;, &lt;stdlib.h&gt;, &lt;time.h&gt;.

**How the Code Works**

- **Input**: Most scripts prompt the user for input via cin, such as entering numbers for mathematical operations or names for string processing.
- **Processing**: The code executes logic based on the specific file-performing calculations (e.g., average, factorial), managing object states, or running game loops.
- **Output**: Results are displayed to the console using cout.

**Usage**

To run any of the programs, ensure you have a C++ compiler (like g++) installed.

- **Clone the repository**:

Bash

git clone <https://github.com/ahmadhassan-prodev/learn-cpp-programming.git>

cd learn-cpp-programming

- **Compile a specific file**:

Bash

g++ filename.cpp -o output

- **Run the executable**:

Bash

./output

**Learning Outcomes**

By studying this repository, one can learn:

- How to manage user input and console output effectively.
- The difference between pass-by-value and using pointers for memory efficiency.
- How to encapsulate data and behavior within classes.
- Algorithm basics, such as primality testing, leap year calculation, and generating random numbers.
